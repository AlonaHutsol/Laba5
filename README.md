# Лабораторна робота №5  
## Тема: Використання структур у C++

### Умова задачі
Структура містить такі поля:
- прізвище та ім’я студента
- курс
- група
- оцінки з 4 предметів

Потрібно відібрати студентів, які мають менше **50 балів хоча б з одного екзамену**, та визначити їхню кількість.

---

# Алгоритм роботи програми

1. Оголосити структуру `Student`, яка містить:
   - `name` – прізвище та ім’я студента  
   - `course` – курс  
   - `group` – група  
   - `mark1`, `mark2`, `mark3`, `mark4` – оцінки з чотирьох предметів  

2. У головній функції `main()`:
   - оголосити змінну `n` – кількість студентів;
   - створити масив структур `Student st[100]` для збереження інформації про студентів;
   - оголосити змінну `count = 0` для підрахунку студентів, які відповідають умові.

3. Ввести кількість студентів `n`.

4. За допомогою циклу `for` ввести дані про кожного студента:
   - прізвище та ім’я
   - курс
   - групу
   - 4 оцінки.

5. Виконати **лінійний пошук** по масиву студентів.

6. Для кожного студента перевірити умову:
   - якщо хоча б одна оцінка **менша за 50**, студент задовольняє умову.

7. Якщо умова виконується:
   - вивести дані студента;
   - збільшити змінну `count`.

8. Після перевірки всіх студентів вивести:
   - список студентів з оцінками < 50;
   - загальну кількість таких студентів.

---
